## 🚀 Sobre o projeto

Este site foi desenvolvido com uma abordagem moderna de desenvolvimento assistido por IA (AI-assisted development), utilizando engenharia de prompts para acelerar a criação, otimização e refinamento da interface.

A construção combinou:
- Estrutura manual em HTML/CSS
- Refinamento com modelos de IA (Claude, ChatGPT)
- Iterações focadas em UX, performance e conversão

O objetivo foi criar uma experiência premium, rápida e altamente otimizada para conversão.
